Overview

The NAB Mazes subtest is a neuropsychological assessment tool designed to evaluate visuospatial problem-solving abilities, planning capacity, and executive functioning. This test is part of the broader Neuropsychological Assessment Battery and is commonly used in cognitive assessments for various clinical populations.

Test Description

The NAB Mazes requires participants to navigate through increasingly complex maze puzzles from start to finish. The assessment measures:

  • Visuospatial processing: Ability to perceive and analyze visual spatial relationships
  • Planning and organization: Capacity to plan and execute complex sequences of actions
  • Problem-solving: Ability to find solutions to novel challenges
  • Cognitive flexibility: Adaptability to changing task demands
  • Working memory: Holding and manipulating spatial information during task performance

Administration

  • Time Limit: Typically 3-5 minutes per maze, with total administration time of approximately 15-20 minutes
  • Materials: Maze stimulus booklets, response sheets, timer
  • Instructions: Participant must find the correct path from the start point to the exit without lifting the writing instrument or retracing steps

Scoring

  • Primary Score: Total number of mazes completed correctly within time limits
  • Secondary Measures: Time to completion, error rates, strategy use
  • Age-adjusted norms: Standardized scores based on age-referenced normative data

Clinical Applications

  • Traumatic brain injury (TBI) assessment
  • Multiple sclerosis cognitive evaluation
  • Schizophrenia and other psychotic disorders
  • Age-related cognitive decline
  • Neurodevelopmental disorders

Integration with MCCB

The NAB Mazes complements other MCCB assessments by providing additional measures of executive functioning and visuospatial processing, contributing to a comprehensive cognitive profile.
