Brief Visuospatial Memory Test-Revised (BVMT-R)

Test Overview

Purpose: Measure visuospatial memory and learning abilities

Age Range: Adults 18-79 years

Administration Time: 45 minutes (includes 25-minute delay)

Format: Individual administration

Author: Ralph H. B. Benedict, PhD, ABCN (1997)

Test Description

The BVMT-R is a widely used neuropsychological assessment tool designed to measure visuospatial memory and learning. It assesses an individual's ability to learn, remember, and recognize geometric figures presented visually. The test is particularly useful for evaluating patients with head trauma, learning disabilities, psychiatric disorders, and mixed diagnoses.

Administration Procedure

  1. Learning Trials (3 trials):
    • Patient views 6 geometric figures arranged in a 2×3 array for 10 seconds
    • Immediately after, patient draws as many figures as possible in correct positions
    • Process is repeated for 3 trials
  2. Delay Period (25 minutes):
    • Patient engages in other activities (preferably verbal tasks)
    • No exposure to the test figures during delay
  3. Delayed Recall:
    • Patient draws the 6 figures from memory
    • Same procedure as learning trials
  4. Recognition Test:
    • Patient views 12 figures (6 original + 6 foils)
    • Must identify which 6 were in the original stimulus
  5. Optional Copy Trial:
    • Patient copies the figures while viewing them
    • Helps screen for visuoconstructive deficits

Scoring System

Recall Scoring (0-6 points per trial):

  • 2 points: Figure drawn correctly and in correct position
  • 1 point: Figure drawn correctly but in wrong position, OR figure drawn incorrectly but recognizable in correct position
  • 0 points: Figure not drawn or unrecognizable

Recognition Scoring (0-6 points):

  • 1 point: Correct identification of original figure
  • 0 points: Incorrect identification or omission

Derived Scores:

  • Total Recall: Sum of 3 learning trial scores (0-18)
  • Learning Slope: Improvement from Trial 1 to Trial 3
  • Retention Rate: (Delayed Recall / Average Learning) × 100%
  • Recognition Discriminability: Hits minus false alarms

Technical Information

Reliability:

  • Internal consistency: .96-.97 for learning trials, .97 for total recall
  • Test-retest reliability: .60 (Trial 1) to .84 (Trial 3)
  • Delayed recall reliability: .97

Validity:

  • Strong correlation with other visual memory tests
  • Weak correlation with verbal memory tests (good discriminant validity)
  • Sensitive to right hemisphere dysfunction and visuospatial deficits

Clinical Utility:

  • Criterion measure of visuospatial memory in neuropsychological batteries
  • Screening measure in brief cognitive assessments
  • Repeat measure to document changes over time
  • Useful for tracking recovery from brain injury

Normative Data

Standardized on 588 normal participants including:

  • 171 college students
  • 417 community respondents
  • Age range: 18-79 years

Demographically corrected normative scores are available for:

  • Age-adjusted norms
  • Education-adjusted norms
  • U.S. census-matched sample norms

Interpretation Guidelines:

  • Scores 1-2 SD below mean = Mild impairment
  • Scores >2 SD below mean = Moderate-severe impairment
  • Consider demographic factors in interpretation

Clinical Applications

The BVMT-R is particularly useful for assessing:

  • Traumatic Brain Injury: Right hemisphere damage, visuospatial deficits
  • Learning Disabilities: Nonverbal learning disorders
  • Psychiatric Disorders: Schizophrenia, depression, bipolar disorder
  • Neurodegenerative Diseases: Early Alzheimer's, vascular dementia
  • Stroke: Right hemisphere CVA
  • Multiple Sclerosis: Cognitive impairment monitoring
